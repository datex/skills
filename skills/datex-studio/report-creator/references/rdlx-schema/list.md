# RDLX-JSON — list

List report item (data-bound repeating container).

## Definitions

### List

Represents properties for a List in a report

| Property | Type | Req | Description |
| --- | --- | --- | --- |
| `Name` | `string` | **Y** | A unique identifier for a List |
| `Type` | `"list"` | **Y** | Should be set to "list" |
| `Action` | → Action |  | An action (such as a hyperlink) associated with a List |
| `KeepTogether` | `boolean` |  | Whether the entire contents of a List should be kept together on one Page if possible |
| `ZIndex` | `number` |  | The stack order of a List |
| `Visibility` | → Visibility |  | Whether a List is hidden |
| `ToolTip` | `string` |  | The text displayed when a report reader hovers over a List at preview time |
| `Bookmark` | `string` |  | The Id of a List for a Jump To Bookmark interactivity action. |
| `Label` | `string` |  | The text associated with a List that appears in the Report Map and Table of Contents |
| `SortExpressions` | → SortExpression[] |  | A collection of expressions by which to sort repeated list regions |
| `ConsumeWhiteSpace` | `boolean` |  | Whether all whitespace in a List should be consumed when contents grow |
| `PageBreak` | `string` |  | The location of a page breaks generated by a List |
| `NewPage` | `string` |  |  |
| `Filters` | → Filter[] |  | A collection of filters to be applied to the data for each row of a List |
| `NewSection` | `boolean` |  | A List has its page numbering |
| `Group` | → Grouping |  | The expressions by which to group the data in a List |
| `RowsOrColumnsCount` | `number` |  | Number of rows or columns (depends on the GrowDirection value) |
| `GrowDirection` | `string` |  | List layout grow direction |
| `ReportItems` | → ReportItem[] |  | A collection of the report items within a List |
| `NoRowsMessage` | `string` |  | Text to render instead of the list layout when no rows of data are available for a List. |
| `DataSetName` | `string` |  | The name of the dataset to use to bind data to a List. |
| `Left` | `string` |  | A value in Length units indicating the distance of a List from the left of the list's container |
| `Top` | `string` |  | A value in Length units indicating the distance of a List from the top of the list's container |
| `Width` | `string` |  | A value in Length units indicating the width of a List |
| `Height` | `string` |  | A value in Length units indicating the height of a List |
| `FixedWidth` | `string` |  | A value in Length units indicating the fixed width of a List |
| `FixedHeight` | `string` |  | A value in Length units indicating the fixed height of a List |
| `OverflowName` | `string` |  | The name of the overflow place holder for a fixed List layout. |
| `StyleName` | `string` |  |  |
| `Style` | object |  | Represents style information for a List |

**Style properties:**

Includes **FullTextStyle** properties.
