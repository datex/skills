# RDLX-JSON — banded-list

BandedList report item and sub-definitions (BandedListHeader, BandedListFooter, BandedListGroup).

## Definitions

### BandedList

Represents properties of a Banded List in a report

| Property | Type | Req | Description |
| --- | --- | --- | --- |
| `Name` | `string` | **Y** | A unique identifier for a Banded List |
| `Type` | `"bandedlist"` | **Y** | Should be set to "bandedlist" |
| `KeepTogether` | `boolean` |  | Whether the entire contents of a Banded List should be kept together on one Page if possible |
| `ZIndex` | `number` |  | The stack order of a Banded List |
| `Visibility` | → Visibility |  | Whether a Banded List is hidden |
| `ToolTip` | `string` |  | The text displayed when a report reader hovers over a Banded List at preview time |
| `Bookmark` | `string` |  | The Id of a Banded List for a Jump To Bookmark interactivity action. |
| `Label` | `string` |  | The text associated with a Banded List that appears in the Report Map and Table of Contents |
| `ConsumeWhiteSpace` | `boolean` |  | Whether all whitespace in a Banded List should be consumed when contents grow |
| `PageBreak` | `string` |  | The location of a page breaks generated by a Banded List |
| `NewPage` | `string` |  |  |
| `Filters` | → Filter[] |  | A collection of filters to be applied to the data for each row of a Banded List |
| `NewSection` | `boolean` |  | A Banded List has its page numbering |
| `DataSetName` | `string` |  | The name of the dataset to use to bind data to a Banded List. |
| `PreventOrphanedFooter` | `boolean` |  | Whether a lonely Banded List footer should be prevented. |
| `Header` | → BandedListHeader |  | The collection of header bands in a BandedList. |
| `Footer` | → BandedListFooter |  | The collection of footer bands in a BandedList. |
| `Details` | object |  | The collection of detail bands in a Banded List. |
| `PreventOrphanedHeader` | `boolean` |  | Whether a lonely Banded List header should be prevented. |
| `Groups` | → BandedListGroup[] |  | The collection of banded list groups consisting of group expressions, group headers, and group footers for a Banded List. |
| `Left` | `string` |  | A value in Length units indicating the distance of a Banded List from the left of the banded list's container |
| `Top` | `string` |  | A value in Length units indicating the distance of a Banded List from the top of the banded list's container |
| `Width` | `string` |  | A value in Length units indicating the width of a Banded List |
| `Height` | `string` |  | A value in Length units indicating the height of a Banded List |
| `FixedWidth` | `string` |  | A value in Length units indicating the fixed width of a Banded List |
| `FixedHeight` | `string` |  | A value in Length units indicating the fixed height of a Banded List |
| `OverflowName` | `string` |  | The name of the overflow place holder for a fixed Banded List layout |

**`Details` properties:**

| Property | Type | Req | Description |
| --- | --- | --- | --- |
| `ReportItems` | → ReportItem[] |  | A collection of reports item that are contained within the Details band of a Banded List |
| `Name` | `string` |  | A unique identifier for the Details band |
| `Action` | → Action |  | An action (such as a hyperlink) associated with a Details |
| `CanShrink` | `boolean` |  | Whether the height of the Details band can decrease to match its contents |
| `CanGrow` | `boolean` |  | Whether the height of the Details band can increase to match its contents |
| `KeepTogether` | `boolean` |  | Whether the entire contents of the Details band should be kept together on one Page if possible |
| `Height` | `string` |  | A value in Length units indicating the height of the Details band in a Banded List |
| `Visibility` | → Visibility |  | Whether the Details band is hidden |
| `PageBreak` | `string` |  | The location of a page breaks generated by the Details band |
| `NewPage` | `string` |  |  |
| `CustomProperties` | object[] |  |  |
