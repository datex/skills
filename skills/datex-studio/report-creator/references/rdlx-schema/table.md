# RDLX-JSON — table

Table report item and sub-definitions (TableRow, TableCell, TableColumn, TableGroup, TableHeader, TableFooter, etc.)

## Definitions

### Table

Represents properties for a Table in a report

| Property | Type | Req | Description |
| --- | --- | --- | --- |
| `Name` | `string` | **Y** | A unique identifier for a table |
| `Type` | `"table"` | **Y** | Should be set to "table" |
| `KeepTogether` | `boolean` |  | Whether the entire contents of a Table should be kept together on one Page if possible |
| `ZIndex` | `number` |  | The stack order of a Table |
| `Visibility` | → Visibility |  | Whether a Table is hidden |
| `ToolTip` | `string` |  | The text displayed when a report reader hovers over a Table at preview time |
| `Bookmark` | `string` |  | The Id of a Table for a Jump To Bookmark interactivity action. |
| `Label` | `string` |  | The text associated with a Table that appears in the Report Map and Table of Contents |
| `SortExpressions` | → SortExpression[] |  | A collection of expressions by which to sort repeated table rows |
| `PageBreak` | `string` |  | The location of a page breaks generated by a Table |
| `NewPage` | `string` |  |  |
| `Filters` | → Filter[] |  | A collection of filters to be applied to the data for each row of a Table |
| `NewSection` | `boolean` |  | A Table has its page numbering |
| `NoRowsMessage` | `string` |  | Text to render instead of the list layout when no rows of data are available for a Table. |
| `DataSetName` | `string` |  | The name of the dataset to use to bind data to a Table. |
| `FrozenRows` | `number` |  | A value indicating how many rows should remain visible while scrolling a long table in the report viewer that operates in Galley mode |
| `FrozenColumns` | `number` |  | A value indicating how many columns should remain visible while scrolling a wide table in the report viewer that operates in Galley mode |
| `RepeatToFill` | `boolean` |  | Whether a fixed table layout should fill the unused space with empty table rows |
| `RepeatBlankRows` | `string` |  |  |
| `PreventOrphanedFooter` | `boolean` |  | Whether a lonely table footer should be prevented. |
| `TableColumns` | → TableColumn[] |  | The collection of columns in a Table. |
| `Header` | → TableHeader |  | The collection of header rows in a Table. |
| `TableGroups` | → TableGroup[] |  | The collection of table groups consisting of group expressions, group headers, and group footers for a table. |
| `Footer` | → TableFooter |  | The collection of footer rows in a Table. |
| `Details` | → TableDetails |  | The collection of detail rows for a Table. |
| `Left` | `string` |  | A value in Length units indicating the distance of a Table from the left of the table's container |
| `Top` | `string` |  | A value in Length units indicating the distance of a Table from the top of the table's container |
| `Width` | `string` |  | A value in Length units indicating the width of a Table |
| `Height` | `string` |  | A value in Length units indicating the height of a Table |
| `FixedWidth` | `string` |  | A value in Length units indicating the fixed width of a Table |
| `FixedHeight` | `string` |  | A value in Length units indicating the fixed height of a Table |
| `OverflowName` | `string` |  | The name of the overflow place holder for a fixed Table layout |
| `StyleName` | `string` |  |  |
| `MaxDetailsPerPage` | `string` |  |  |
| `PreventOrphanedHeader` | `boolean` |  |  |
| `Style` | object |  | Represents style information for a Table |

**Style properties:**

Includes **FullTextStyle** properties.

### TableCell

Represents the contents of a cell in a Table.

| Property | Type | Req | Description |
| --- | --- | --- | --- |
| `Item` | → ReportItem | **Y** | A report item that is contained within a cell of a Table |
| `ColSpan` | `number` |  | The number of columns that the cell spans. |
| `RowSpan` | `number` |  | The number of rows that the cell spans. |

### TableDetails

Represents the details rows for a Table

| Property | Type | Req | Description |
| --- | --- | --- | --- |
| `TableRows` | → TableRow[] |  | A collection of the details rows for a Table |

### TableFooter

Represents the footer rows for a Table

| Property | Type | Req | Description |
| --- | --- | --- | --- |
| `PrintAtBottom` | `boolean` |  | Whether a table footer always prints at the bottom of the page |
| `TableRows` | → TableRow[] |  | A collection of the footer rows for a Table |
| `RepeatOnNewPage` | `boolean` |  | Whether rendered footer rows are repeated on every Page where the header’s parent Table is rendered |

### TableRow

Represents a row of cells in a table

| Property | Type | Req | Description |
| --- | --- | --- | --- |
| `Action` | → Action |  | An action (such as a hyperlink) associated with a TableRow |
| `Height` | `string` |  | A value in Length units indicating the height of a row |
| `Visibility` | → Visibility |  | The presence of a row in a Table. |
| `TableCells` | → TableCell[] |  | The contents of a row in a Table. |
| `CustomProperties` | object[] |  |  |
| `RepeatAsBlankRow` | → ThreeStateBoolean |  | Indicates that the row style should be used for filling the blank (Auto is true) |
