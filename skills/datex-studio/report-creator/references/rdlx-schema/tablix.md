# RDLX-JSON — tablix

Tablix (matrix/crosstab) report item and sub-definitions (TablixBody, TablixHierarchy, TablixMember, TablixGrouping, etc.)

## Definitions

### Tablix

Represents properties for a Tablix in a report

| Property | Type | Req | Description |
| --- | --- | --- | --- |
| `Name` | `string` | **Y** | A unique identifier for a Tablix |
| `Type` | `"tablix"` | **Y** | Should be set to "tablix" |
| `KeepTogether` | `boolean` |  | Whether the entire contents of a Tablix should be kept together on one Page if possible |
| `ZIndex` | `number` |  | The stack order of a Tablix |
| `Visibility` | → Visibility |  | Whether a Tablix is hidden |
| `ToolTip` | `string` |  | The text displayed when a report reader hovers over a Tablix at preview time |
| `Bookmark` | `string` |  | The Id of a Tablix for a Jump To Bookmark interactivity action. |
| `Label` | `string` |  | The text associated with a Tablix that appears in the Report Map and Table of Contents |
| `SortExpressions` | → SortExpression[] |  | A collection of expressions that are applied to the filtered data of a Tablix to order the data. |
| `PageBreak` | `string` |  | The location of a page breaks generated by a Tablix |
| `NewPage` | `string` |  |  |
| `Filters` | → Filter[] |  | A collection of filters to be applied to the data for each row of a Tablix |
| `NewSection` | `boolean` |  | A Tablix has its page numbering |
| `NoRowsMessage` | `string` |  | Text to render instead of the tablix layout when no rows of data are available for a Tablix. |
| `DataSetName` | `string` |  | The name of the dataset to use to bind data to a Tablix. |
| `FrozenRows` | `number` |  | A value indicating how many rows should remain visible while scrolling a long tablix in the report viewer that operates in Galley mode |
| `FrozenColumns` | `number` |  | A value indicating how many columns should remain visible while scrolling a wide table in the report viewer that operates in Galley mode |
| `RepeatColumnHeaders` | `boolean` |  | Whether tablix column headers are repeated on each page where the column traverses a page break. |
| `RepeatRowHeaders` | `boolean` |  | Whether Tablix row headers are repeated on each page where the row traverses a page break. |
| `GroupsBeforeRowHeaders` | `number` |  | The count of dynamic member columns of the first Tablix column group to render before the tablix row header. |
| `LayoutDirection` | `string` |  | The layout direction of Tablix columns. |
| `RowHierarchy` | → TablixHierarchy |  | The row hierarchy of a Tablix. |
| `ColumnHierarchy` | → TablixHierarchy |  | The column hierarchy of a Tablix. |
| `Corner` | → TablixCornerRow[] |  | The corner of a Tablix. |
| `Body` | → TablixBody |  | The body of a Tablix. |
| `Left` | `string` |  | A value in Length units indicating the distance of a Tablix from the left of the tablix's container |
| `Top` | `string` |  | A value in Length units indicating the distance of a Tablix from the top of the tablix's container |
| `Width` | `string` |  | A value in Length units indicating the width of a Tablix |
| `Height` | `string` |  | A value in Length units indicating the height of a Tablix |
| `FixedWidth` | `string` |  | A value in Length units indicating the fixed width of a Tablix |
| `FixedHeight` | `string` |  | A value in Length units indicating the fixed height of a Tablix |
| `OverflowName` | `string` |  | The name of the overflow place holder for a fixed Tablix layout. |
| `StyleName` | `string` |  |  |
| `Style` | object |  | Represents style information for a Tablix |

**Style properties:**

Includes **FullTextStyle** properties.

### TablixGrouping

Represents expressions by which to categorize data and properties of the resulting data categorization in a Tablix

| Property | Type | Req | Description |
| --- | --- | --- | --- |
| `PrintFooterAtBottom` | `boolean` |  | Whether a group footer always prints at the bottom of the page |
| `Name` | `string` |  | A unique identifier for a TablixGrouping. |
| `DocumentMapLabel` | `string` |  | The text associated with a Tablix Group instance that appears in the Report Map and Table of Contents |
| `GroupExpressions` | `string`[] |  | A collection of expressions of a Tablix Group. |
| `PageBreak` | `string` |  | The location of a page breaks generated by a Tablix Group. |
| `PageBreakDisabled` | `string` |  | Whether the generated page break should be disabled. The value or the expression result should be either "true" or "false" |
| `NewPage` | `string` |  |  |
| `Filters` | → Filter[] |  | A collection of filters for a Tablix Group. |
| `ParentGroup` | `string` |  | An Id of a parent group |
| `NewSection` | `boolean` |  | A Tablix Member has its page numbering |
| `Enabled` | `string` |  |  |

### TablixMember

Represents the structure of an item in a Tablix hierarchy.

| Property | Type | Req | Description |
| --- | --- | --- | --- |
| `Header` | → TablixHeader |  | Gets os sets a cell of a row header or a column header. |
| `BodyIndex` | `number` |  |  |
| `BodyCount` | `number` |  |  |
| `Visibility` | → Visibility |  | The presence of a TablixMember element. |
| `KeepTogether` | `boolean` |  | Whether the entire contents of a TablixMember should be kept together on one Page if possible |
| `Group` | → TablixGrouping |  | The group for a TablixMember element. |
| `KeepWithGroup` | `string` |  | Whether an attempt is made to keep each rendered instance of a parent static member with the closest non-hidden instance of a preceding or subsequent sibling dynamic member. |
| `RepeatOnNewPage` | `boolean` |  | Whether instances of rendered static members are repeated on every Page on which at least one complete instance of the associated dynamic member is rendered. |
| `SortExpressions` | → SortExpression[] |  | A collection of expressions that determine the order of the TablixMember data. |
| `Children` | → TablixMember[] |  | A collection of child tablix members for the current member. |
